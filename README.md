# Strava Full Year Calendar

A clean, beautiful HTML calendar that visualizes your entire year of Strava workouts.

## Features

- Full-year calendar view (all 12 months)
- Each workout day shows:
  - Activity emoji (🏃‍♂️ 🚴 🏊 etc.)
  - Total miles for that day
  - State abbreviation (e.g. CA)
- Clean, modern dark sporty design
- Responsive layout (works well on desktop and mobile)
- Summary with total workout days, total miles, and **average minutes per workout day**
- Loads credentials securely from `config.json`

## How to Use

1. Clone or download the repository
2. Run the script once:
   ```bash
   python3 StravaCalendarReview.py
